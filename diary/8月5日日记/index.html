# 学习

# 黄金批次

  • 这两天有点浪费时间，划水很严重。今天在摸鱼的时候晓蝶姐让我做一个生物发酵（青霉素）黄金批次的数据分析，就是找到影响它产物浓度的因素，并且找到一些点进行调整，让最后的产率提高。
  • 第一题是找出与产物浓度最相关的特征，这个比较简单，画了一个相关性热力图就出来了
  • 第二题让找调整点。不会完全。 模型想用随机森林（其实内容全是 AI），最后也没做出来
  • 上班摸鱼挺抖音自己收藏的音乐合集，完全听一遍需要一个小时。

# 鸢尾花

  • 姐又让我做了一个更简单的，鸢尾花分类
  • 第一步把数据集分成 8：2 的训练集和测试集，这个简单
  • 第二步建立模型并预测，我告诉 AI 用 svm，AI 很快搭出来了，准确率到了 97% 左右
  • 但是我在这个过程中好像什么都没学到。
  • 晓蝶姐让我自己把代码照着 csdn 上的写一遍，这样在过程中更深地理解那些函数都在干什么

# 书记

  • 了解到书记的实习，被分配了一些杂活，做了一个数据表有关的前端网页，挺厉害的，用到了 sql，还是得学学这个的

# 生活

  • 昨天晚上十二点多一点点睡得，感觉还可以，白天没有特别困
  • 吃的非常混乱。连续三晚上吃汉堡可乐了吧。。主要不知道别的吃点什么好，好像也没有非常好吃的

# 体育课

  • 抢课抢到给分最高的武术了，很开心。我身体应该算协调的吧。

# 游戏

  • 给火影冲了个 25 的忍法帖，以后可以每个忍法帖无限续杯了。
  • 感觉现在的忍者基本山够玩了，old 神力这一块。高招循环吧，出了新的不秒，等变 old 再慢慢品

# 博客

  • 网站不知道怎么的自己就显示图片了，非常的惊喜。可能是因为我写的内容太多了吧，触发什么机制了？（纯瞎猜）
  • 犯懒了。。这几天回宿舍就是疯狂刷视频。然后完全没有干活。
