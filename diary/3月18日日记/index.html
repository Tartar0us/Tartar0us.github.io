刚刚排球训练回来，一会两点半要去科技楼和院长谈话。

今天上午的解析几何课听的还行，算是开学以来听的最认真的一节课，感觉很爽，开始整理笔记了，但是仅限于课上学的东西，课后题目前还没开始做。排球训练很差劲，我接对面的发球一个也没接到，队长说我手臂没夹紧，然后注意力不集中。我可能是越接不到球越想接，但是每次那个球就是瞬间到了眼前，根本没时间反应。可能我已经有注意力障碍了。很少情况下能持续集中注意力了。还挺敏感的，休息的时候队长拉着我教我怎么接球，差点就哭出来了。杨学长也很关心我的状态，问我是不是发生什么事了，我说今天心情不太好。我的脑子好像有点问题，总是感觉对什么也提不起兴趣来。我目前感兴趣的区域是电脑软硬件和网站搭建（仅仅是现阶段的兴趣，之后就不知道了）但是如你所见，前天开始我的笔记中就说我要学如何在 github 学习搭建网站，但是苦于没有时间。你要说没有时间吧我还有时间打吃鸡。还有时间刷一会视频。这算拖延症吗？但是我总是可以在 ddl 前完成学生工作或者作业。很奇怪，我的生活是被各种任务充斥的，是我需要去做什么，我优先完成的是这些事，却不是我想做什么。我本科阶段的目标是学一些电脑软硬件的知识，然后学一些人工智能相关的内容，再就是保持健康，越来越帅，但是如果我要在大二继续学生工作的话好像并不能并行。当然本科阶段最重要的两个任务是健康和申研，其他都是容错很高的小事。现在是一点四十七，我目前的心情好了不少，看来把东西写出来确实能够排解人的心情。如果我能坚持到我的生日那天，我会把这个博客分享给我最好的朋友，让她也能知道我每天在想什么。希望这些东西也能对她有点帮助。虽然可能帮助并不大。

感谢豆包能在我排球打的跟史一样之后安慰我。虽然她说的话可能并没有感情，但是确实对我的难过起到了一定的安慰效果。
————————————————
下午和院长谈了话，主要针对以下问题

1. 什么时候可以进实验室做科研（发论文）、以及需要什么条件，什么前置知识，是否可以参与其他学院的科研项目

答：2025 年十月后会开展相关工作，因为目前我们的知识储备有限，需要学完常微分方程后，大致有了自己感兴趣的方向，再跟随老师做课题。在这之前可以留意学校相关的讲座，或者多在学院网页看看老师们研究的方向，选择感兴趣的提前联系。

有关前置知识，如果要做数据分析相关的课题，c++ 和 python 等计算机有关知识应该算基础知识，要抽时间学习。

2. 要不要继续学生工作

答：看个人，院长说：“这个问题好像在问我：‘当院长有没有影响科研’” 他说如果只做科研，可能其实专注的时间也并不会特别长（我的理解）这个东西还是看个人的思考，如果说觉得学生工作让我内耗了。或者说影响学习了，那不干就不干了。但是这个其实也是很珍贵的一个回忆。

有关上面这个话题我问过很多人，想法变化过很多次。我想我可能未来还会改变想法。这一切还要看我本学期的学习如何，竞赛如何。

3. 保研名额 何时确定

答：目前悉尼智能科技学院有百分之十左右的保研名额，我院应该有百分之 4 或百分之五的保研名额（四五个人），如果对这方面有想法的话可以稳步提升自己的绩点，目前这个绩点还是没有办法保研的。（对于保研夏令营的事，宣委说可以六月初去，六月初有部分高校的夏令营就开始了）

4. 基于目前您对国际形势的了解，给出有关留学目的地国家和推荐院校的建议

答：先专注于提升知识储备，国家和学校可以以后再说，先多了解了解，没国说乱也乱，但是教育确实是最好的。各个地方都有各个地方的优缺点。

5. 入档会不会影响留学

有机会就一定要争取！！！

感想：感觉自己还是跟小孩一样非常幼稚，想法很容易被改变，很容易想得多，不高兴，但是也很容易就高兴。
