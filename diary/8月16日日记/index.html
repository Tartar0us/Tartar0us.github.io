# 生活

感觉周末最能写的就是生活这一部分了。基本周末的时候不咋学习，游戏也懒得打，主要就是做做火影日常，然后刷视频。刷视频很浪费时间，本来可以做好多事 
文本如果前面加上一个tab会变成奇怪的形式，但是不知道预览效果有没有区别，应该差不多，就是格式更规整了一些
今天可算约上高中的几个同学出去玩了，非常的爽，十点多到了万达，喝了阿水的一个蓝色气泡水，忘了叫啥名字了，挺好喝，然后十三块钱，量特别大。
 下午去打台球了，许东晨打的是我们几个人里边最厉害的。
北丐好吃捏，人均36，打台球三个小时人均7块，爽玩。
玩完四点，回家快五点，呆了半个小时就去晚上的场了，两场撞了，妈妈那边组的局介绍了一个中农研二的哥，还有一个之前新一中的校友张少博，我就说很眼熟来着，确实见过。物化政，和李永震很熟。

# 学习

今天出六级了，440分擦边过，有点意思哈，跟我自己估分差不多。还需要继续努力。网上说浙大的保研要求六级460分，清北的要求550，我想刷分刷到550就停，因为这就很够用了，考雅思应该也行了。但是雅思还是得学。今天吃饭认识了一个香港理工大学的姐姐，学的地理信息专业，自学雅思考了7.5的高分。很厉害，小分最低的是6.5 ,向她学习，争取申请更好的研究生。
感觉辅修了计算机之后说不定可以直接跨申计算机了。学的东西还是挺有意思的。但是不知道算法市场有没有饱和。感觉这个以后还得是AI来做。应该在业余时间学习有关商学和管理学的知识，为以后做准备。如果毕业之后有技术风口，可以抓住，搞个团队一起创业，挣快钱。
下周开始就是我第五周的实习了。感觉时间过的有点太快了。明天早起准备一下实习证明盖章之类的文件，然后还有就是要趁早弄一下三下乡的b站账号，运营的不是很好，流量很难看。目前还有半个月就要答辩了。哦对还没有做答辩ppt。这个应该还是要拜托fqq帮帮忙。
明天抽空整理一下财务情况。感觉有点经济崩溃了。。

# KAGGLE

好像好久没动了。目前还是跟个新兵蛋子一样。需要通过系统的学习来熟悉比赛的流程。
哦卧槽，下周还有一个数学建模校赛要搞。

# 好玩的

  • 本来想自己搞一个台式机在学校玩，想到是一个可能时间不是很多，第二个预算不太够，现在花呗还欠着不少，每个月可能很难匀出来。先算了。但是又想到准备收同学一个 450 的显示器，450 的电脑能买一个非常好的了。仔细想想感觉这个事还要考虑。
  • 这几天没事看看图拉丁吧的视频，加点钱 500 块钱能买到啥样的电脑。
  • 要是能玩三角洲和打瓦，就借给舍友玩
  • 这个设想是基于昨天树莓派的设想，今天问了一下哈工大玩嵌入式的同学， 他买的树莓派 5 代花了五百多块钱，感觉有这个钱完全可以自己搭一个小电脑了。就是功率绝对是一个问题。
  • 米罗礽绝对是一个天才，大一就自己组装了一台台式，两千多块钱，玩游戏什么的完全都够了。
  • 我想买同学的显示器主要是感觉大屏更加护眼。
  • 桌面的解决方案也想到了，就是买一个倒挂的显示器机械臂，这样就可以了。但是总体下来成本来到了 550 元档。实际还是不如自己搭一个主机有意思。这样可能自己更多的时间捣鼓电脑，就不会浪费时间在没有用的东西上了。
  • 我觉得捣鼓一个电脑更多的是可以办公和学习电脑相关技术，到时候游戏性能其实就成了可有可无的东西
