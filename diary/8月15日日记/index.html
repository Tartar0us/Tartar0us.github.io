# 学习

  • SQL 又学了一点，当真是越往后越难啊。
  • 感觉脑子可能是知识过了之后又出去了，很少的东西留下
  • 上午捣鼓了有一个小时装 MySQL。以后一定仔细看教程，不能擅作主张，这样反而会多很多事

# 生活

  • 五点到德州，打车回家被说了两句。我就是不想坐公交车。我认为是观念不同，20 多块钱又不是很多，我又不是天天花，为了不等公交在外边热的受罪，我觉得这 20 多块钱花的挺值的。以后要是从车站回家，我还会打车。不会跟家长汇报了。
  • 晚上有点困，妈妈拉着去奥莱看了看五八九街区，说实话就是模仿的唐山宴，并没有唐山宴那么大，主要还是有一些德州自己的城市记忆。在那里看到了好多有关德州历史的东西，比如德州一中和二中的一些历史毕业照片，很多年之前校门的样子等等。我个人对这些历史的东西并不是很感兴趣。我认为历史那么多，目前的技术，如果让人学习的话，一辈子也学不完，而且社会是进步的，我们焦虑未来和过好当下还很困难的，没有时间去怀念过去。历史学家什么的当然有他存在的意义，但是普通人没必要对历史有太多太深的研究
  • 上面的理论是比较主观的观点。我刚才还想从自己内心的角度论证一下为什么我是这么想的，但是发现自己的逻辑好像并不是很完整。但是我自己对一些旧的东西不是很感兴趣就是了。（和新的东西相比）
  • 最后走的时候想买一个汽车模型。问妈妈妈妈不给买。上车之后越想越难受哭了。后来过了四五分钟就没事了。感觉自己跟小孩子一样。这也挺好的，省钱了。

# 好玩的东西

  • 火影日常
  • 我发现我现在打字的速度和准度完全不如白天在公司的打字舒畅。经常卡键。。晚上太累了可能是，也可能是有点急于求成了，手指在键位之间转变的太快，跟不上，有时候会按到键位的边缘。
  • 笑死我了，刚才专门去网上查了一下 “键盘上单个键位的学名叫什么” 感觉脑子瓦特了，上一条把他们称作键位，其实就是叫 “按键”。
  • 刷视频刷到树莓派，了解了一下发现是一个非常有意思的东西，相当于一个微型电脑，可以搞很多有意思的东西，其中一个大优点就是非常的便携。然后就是各种借口一应俱全
  • 今天就先写这么多吧。感觉是这周写的最颠的一篇日记了。
