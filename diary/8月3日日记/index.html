# 周五

  • 周五晚回家了，依旧一个半小时高铁
  • 晚上很累，本来想着回家倒头就睡来着， 但是最后磨磨唧唧一点才睡觉。太晚了。

# 周六

  • 很有意思的一天，晓龙来家里，还有昌大学辉，我们一起编纂老刘语录，整理东区五班的史料，奋斗了一天的结果是老刘语录出来电子版了，然后其他的一些纸质版也全部都扫描完了，非常的完美。我想后面用 gitee 搭建一个国内的网站储存这些史料。可以后期慢慢的弄
  • 如果用网站的话不知道能不能放 pdf，这也是一个考验
  • 晚上去打排球，皇橙，菜的没边了，反应很迟钝，只能拦网拦几个，后面也拦不到了。根本没有一传，同队的大哥都不让我接一传，他们四个接。我真是菜的没边了。。
  • 但是打完球，回家到小区门口 711 喝无糖的魔爪，还有三个小面包，非常的爽啊，魔爪真的时这个世界上最好喝的饮料，技能提供能量，又炸爽，还量不大，一次喝完不会撑得慌

# 周日

  • 躺了一个白天，然后晚上回石家庄，到了酒店之后吃了华莱士和炸鸡，接着加班三下乡的材料到现在他妈快一点。
  • 我得快点睡觉去了。
  • 哦我还美化了一下 windows 桌面，整体看来还不错，
  • ![[Pasted image 20250804003625.png]]
  • 拖到现在还是没写月度总结和年度总结。你真能拖啊。
  • 我要给自己制定新的计划了，大一一年从 j 人变 p 人有点可惜了。
