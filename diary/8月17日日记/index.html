# 日常

中午去吃了李斌叔叔家妹妹的升学宴，吃的非常爽，菜都是硬菜，分量很足，张紫旭一家也来了，挺热闹的。
到了那里之后我才知道李梓希的对象是张紫旭的同学，这关系挺有意思，让我愈发觉得德州很小了。因为昨天晚上出去吃饭，认识的张少博，他妈妈和我爸爸是同事，我和他之前在新校见过几次面，所以昨天晚上我一见面就觉得特别眼熟。后来一问果然是物化政，有点眼缘。
右眼的散光还在加重。

# 高铁奇遇

坐我旁边的阿姨说让我跟她换一下，因为她东西在脚底下感觉空间不够，我说没事，帮她把东西拿到头顶的架子上了。然后就开始跟阿姨搭话，问我去干啥，我说实习，她是来石家庄找女儿，女儿去年刚毕业，在石家庄一个身份证制造中心上班，在山西太原某个大学学的计算机，毕业之后直接考公，很幸运来到这个岗位，工资六千，算是事业编，阿姨说现在工资比她高（阿姨是小学老师，马上快退休了。叔叔是体育老师，189/130kg）姐姐找了一个军校的对象，两个人一个事业编，一个中尉，无敌了这个组合。
一路上跟阿姨聊了很多东西，从高考的不容易，到找工作应该找什么样的，未来应该怎么准备，谈恋爱和结婚应该怎么样。
# 谈恋爱
阿姨说这个不用着急，慢慢遇到有缘分的再说，最好是找一个喜欢自己的，因为喜欢一个人就会主动付出。她特别开明，说准备就跟着女婿那边的习俗，正常商量，对彩礼什么的没有别的要求。然后她也说人的观念是慢慢变的。具体还说什么我忘了，但是我在听的时候自己总结了一下逻辑，我最后的决定是，随缘，不用着急，努力提升自己，后面可能就会遇到更好的人，然后就是，假如你20多岁结婚了，可能就会影响事业，两个人一起努力挣钱还房贷，但是可能会被困在一个小地方，没有出去闯的心气了。但是如果30岁之前一直在努力提升自己，同时在积累财富，说不定到30多岁就可以直接全款买房，到时候圈层也不一样了。
# 努力往高处走
老生常谈，努力入档
# 对于工作
她说如果想挣钱，特别缺钱，就去私企，来钱快，但是累，如果家里不是很缺钱，就去国企，绝对是稳定的选择。她说疫情的时候最轻松的就是老师，最忙的是医生。我说，我之前觉得，努力挣钱，年轻的时候用时间换钱，老了用钱治病，她说我说的有点极端了。我们完全可以折中，不用跟别人比。人类的欲望是无穷无尽的，只要每天吃饱三顿饭就行啊，不用跟别人比。

突然感觉我的博客马上就要十万字了，我觉得是明天

# 竞赛

明天开始打数模。

# 别的事是一点没做。有点太墨迹了。

# 我他妈头有点疼。（物理意义上）

刷到一个视频，挺感慨的，就是视频上写了 up 主从小到大的各个阶段最想要的东西，比如：初二我想要赛尔号里的普尼，初三我想打一辈子 lol 诸如此类，目的应该是回忆自己的童年或者说青春。但是我好像想不太起来我小时候想要的是什么了。很久之前我就很恐惧，当时是因为我想不起来小时候自己的梦想是什么了，自己梦想的职业是什么了。
我太早的丢掉了自己的梦想
