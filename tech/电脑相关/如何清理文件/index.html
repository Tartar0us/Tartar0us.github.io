# 桌面不要放到别的盘

网上有教程教人把桌面放到别的盘里，因为电脑默认把桌面放在c盘，但是在使用过程中，为了方便，我们桌面上的东西会越来越多，c盘因此爆满。
思路是清理c盘，思路是对的，但是做法完全不可取。因为如果你将桌面转移到别的盘里，很可能未来使用某些软件的时候找不到PATH，我的另一个电脑就是这样半报废的。很多软件用不了，或者说配置环境的时候非常麻烦。所以还是老老实实的把桌面放在C盘。或者你有时间折腾一些软件的配置。
