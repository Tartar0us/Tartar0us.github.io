# RNN

  • Recurrent Neural Network（循环神经网络）
  • 有记忆的神经网络，但是记忆太短，容易出错

# LSTM

  • Long Short-Term Memory（长短期记忆）
  • 有三个门来控制记忆，还可以选择删除哪些记忆
  • 遗忘门、输入门、输出门
  • 具有时间顺序
