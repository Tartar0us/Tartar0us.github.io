# 不需要掌握的点

  • 数据写入
  • 数据修改
  • 数据删除
  • 游标
  • 触发器
  • 存储过程
  • 安全管理
  • 角色管理
  • 事务控制
  • 查询速度优化

# 需要掌握的基础部分

  • select 数据提取
  • where 语句使用
  • 表的内联外联
  • 排序和去重
  • SQL 的书写规则
  • 分组 子查询
  • 聚合函数

# 需要掌握的进阶部分

  • case when 的用法
  • 窗口函数如何使用
  • sql 的执行顺序
  • 变量的使用
  • 分组查询严格模式
  • 空值的影响处理
  • count (*) 和 count（列）
  • exist
  • 临时表
  • 自联接
  • 文本时间函数

# sql 三个重点

  • 沟通好，明确需求，避免返工
  • 写好的 sql 语句需要备份
  • sql 是一种语言，mysql 和 livesql 是工具

# 数据分析的主要流程

# 一、数据预处理

1. 对脏数据、空缺数据进行处理，填充或删除（只针对时序数据中的缺失）

# 二、数据分类

  1. 分成两个集，训练集和测试集，一般比例为 8：2，随机分配

# 三、模型建立（还有可能不是模型）

模型是通过经验，让机器吃这些数据，然后预测未来的情况，而另一种办法是直接同数据中计算出公式，未来的预测直接用公式计算解决（这种方法称为机理）

1. 运用各种模型，随机森林，xgboost

# 四、结果分析

  • 精确率，召回率等

  • 实用的数据库工具：Navicat Premium

# 五、可能存在的盲点或信息差

  • ，其语言是 sql，做数据分析最重要的是模型建立的部分，sql 只是一个在数据库中读取数据的工具
